## 一、选择题（每题只有一个正确选项）

1. 【有界性】下列函数中，在$(-\infty,+\infty)$内有界的是（  ）
A\. $f(x)=\frac{x^2}{1+x}$
B\. $f(x)=\frac{1}{1+x^2}$
C\. $f(x)=x\sin x$
D\. $f(x)=e^x$

2. 【单调性】设$f(x)$在$\mathbb{R}$上严格单调递减，则下列函数中一定严格单调递增的是（  ）
A\. $f(x+1)$
B\. $-f(x)$
C\. $f(x^2)$
D\. $|f(x)|$

3. 【奇偶性】设$f(x)$是奇函数，$g(x)$是偶函数，则下列函数中一定为奇函数的是（  ）
A\. $f(x)+g(x)$
B\. $f(x)\cdot g(x)$
C\. $f(g(x))$
D\. $g(f(x))$

4. 【周期性】设函数$f(x)$定义域为$\mathbb{R}$，且对任意$x$满足$f(x+1) = -f(x)$，则$f(x)$的一个周期为（  ）
A\. 1
B\. 2
C\. 3
D\. 4

5. 【综合判断】下列说法正确的是（  ）
A\. 若$f(x)$在区间$(a,b)$内有界，则$f(x)$在$(a,b)$内必连续
B\. 若$f(x)$是奇函数，则必有$f(0)=0$
C\. 若$f(x)$是周期函数，则其导函数也是周期函数
D\. 严格单调函数必有反函数，且反函数与原函数单调性相同

## 二、证明题

6. 【有界性・均值不等式法】证明函数 $f(x)=\frac{x^2}{1+x^4}$ 在 $(-\infty,+\infty)$ 内有界。

7. 【单调性・定义法】设函数$f(x)$在$\mathbb{R}$上满足$f(x+y)=f(x)+f(y)$，且当$x>0$时，$f(x)>0$。证明：$f(x)$在$\mathbb{R}$上严格单调递增。

8. 【奇偶性・赋值法】设函数$f(x)$定义域为$\mathbb{R}$，且对任意实数$x,y$均满足：
$f(xy) = x f(y) + y f(x)$
证明：$f(x)$是奇函数。

9. 【周期性・迭代法】设函数$f(x)$在$\mathbb{R}$上满足 $f(x+a) = \frac{1}{f(x)}\ (a>0,\ f(x)\neq0)$，证明$f(x)$是以$2a$为周期的周期函数。

10. 【周期性・差分累加法】设$f(x)$在$\mathbb{R}$上满足 $f(x) = f(x-1) + \sin\pi x$，证明$f(x)$是以$2$为周期的周期函数。

## 答案与详细解析

### 一、选择题

1. **答案：B**
【解析】

    - A：分母$1+x$在$x=-1$处为 0，$x\to-1$时函数趋向无穷，无界；且分子次数等于分母次数，$x\to\infty$时函数趋向无穷。

    - B：分母$1+x^2$恒正无零点，分母次数（2 次）\> 分子次数（0 次），满足有界条件；事实上$0<\frac{1}{1+x^2}\le1$，有界。

    - C：取$x=2k\pi+\frac{\pi}{2}$，$k\to\infty$时$f(x)\to\infty$，无界。

    - D：$x\to+\infty$时$e^x\to+\infty$，无界。
    【技巧】有理分式有界性可直接用 “分母无零点 \+ 分母次数\> 分子次数” 秒判。

2. **答案：B**
【解析】

    - A：$f(x+1)$是函数图像向左平移 1 个单位，单调性与原函数一致，仍严格递减。

    - B：函数整体加负号，单调性反转。严格递减函数取负后变为严格递增，正确。

    - C：内层$u=x^2$在$(-\infty,0)$递减，在$(0,+\infty)$递增，与递减的外层复合后，$f(x^2)$在$(-\infty,0)$递增，在$(0,+\infty)$递减，非全局单调。

    - D：绝对值会将负函数值翻折，破坏单调性。反例：取$f(x)=-x$，则$|f(x)|=|x|$，不全局递增。
    【技巧】牢记：整体取负、自变量取负都会反转单调性；平移不改变单调性。

3. **答案：B**
【解析】
奇偶运算核心规律：奇 × 偶 = 奇，奇 × 奇 = 偶，偶 × 偶 = 偶。

    - A：奇函数 \+ 偶函数，一般为非奇非偶函数（除非其中一个恒为 0）。

    - B：奇函数 × 偶函数 = 奇函数，正确。

    - C：复合函数$f(g(x))$，内层偶、外层奇，结果为偶函数：$f(g(-x))=f(g(x))$。

    - D：复合函数$g(f(x))$，内层奇、外层偶，结果为偶函数：$g(f(-x))=g(-f(x))=g(f(x))$。

4. **答案：B**
【解析】
迭代代换：已知$f(x+1)=-f(x)$，将$x$替换为$x+1$，得：
$f(x+2) = -f(x+1) = -[-f(x)] = f(x)$
因此周期为 2。
【高频结论】若$f(x+a)=-f(x)$，则$T=2a$是函数的一个周期。

5. **答案：D**
【解析】

    - A：错误。有界推不出连续，例如狄利克雷函数处处有界但处处不连续。

    - B：错误。只有当奇函数在$x=0$处有定义时，才有$f(0)=0$，若定义域不含 0 则不成立。

    - C：错误。只有当函数可导时，周期函数的导函数才是同周期的周期函数，选项未说明可导。

    - D：正确。严格单调函数是一一映射，必有反函数，且反函数与原函数单调性一致。

### 二、证明题

6. **证明**：
对任意实数$x$，由均值不等式 $a^2+b^2\ge2ab$，令$a=1,\ b=x^2$，得：
$1+x^4 \ge 2\cdot1\cdot x^2 = 2x^2$
由于$1+x^4>0$，两边取倒数得：
$\frac{1}{1+x^4} \le \frac{1}{2x^2} \quad (x\neq0)$
对函数取绝对值（函数恒非负）：
$f(x) = \frac{x^2}{1+x^4} \le \frac{x^2}{2x^2} = \frac{1}{2}$
当$x=0$时，$f(0)=0$，显然满足$f(x)\le\frac{1}{2}$。
取$M=\frac{1}{2}$，对所有$x\in\mathbb{R}$都有$|f(x)|\le M$，故$f(x)$在$\mathbb{R}$内有界。
【考点】均值不等式放缩证明有界性

7. **证明**：
任取$x_1,x_2\in\mathbb{R}$，且$x_1<x_2$，则$x_2-x_1>0$。
由题设条件，当$x>0$时$f(x)>0$，因此：
$f(x_2 - x_1) > 0$
利用函数方程$f(x+y)=f(x)+f(y)$，将$f(x_2)$变形拆分：
$f(x_2) = f\left(x_1 + (x_2 - x_1)\right) = f(x_1) + f(x_2 - x_1)$
作差得：
$f(x_2) - f(x_1) = f(x_2 - x_1) > 0$
即$f(x_2) > f(x_1)$，由严格单调递增定义，$f(x)$在$\mathbb{R}$上严格单调递增。
【考点】单调性定义 \+ 柯西方程变形，核心是作差后利用已知条件判断符号

8. **证明**：
第一步：求特殊值$f(0)$
令$x=y=0$，代入方程得：
$f(0) = 0\cdot f(0) + 0\cdot f(0) = 0$

    第二步：求$f(-1)$的值
    令$x=y=1$，得：$f(1) = 1\cdot f(1) + 1\cdot f(1) = 2f(1) \implies f(1)=0$
    令$x=-1,\ y=-1$，得：$f(1) = (-1)f(-1) + (-1)f(-1) = -2f(-1)$
    代入$f(1)=0$，解得$f(-1)=0$。

    第三步：验证奇函数定义
    令$y=-1$，对任意实数$x$代入方程：
    $f(-x) = x\cdot f(-1) + (-1)\cdot f(x)$
    代入$f(-1)=0$，化简得：
    $f(-x) = -f(x)$
    函数定义域为$\mathbb{R}$，关于原点对称，故$f(x)$是奇函数。
    【考点】赋值法证明奇偶性，通过特殊值搭建$f(-x)$与$f(x)$的关系

9. **证明**：
已知对任意$x$，有 $f(x+a) = \frac{1}{f(x)}$。
将等式中所有$x$替换为$x+a$，得：
$f(x+2a) = \frac{1}{f(x+a)}$
将$f(x+a)=\frac{1}{f(x)}$代入上式：
$f(x+2a) = \frac{1}{\frac{1}{f(x)}} = f(x)$
因此$f(x)$是以$2a$为周期的周期函数。
【高频结论】若$f(x+a)=\pm\frac{1}{f(x)}$，则$T=2a$是函数的一个周期。

10. **证明（差分累加法）**：
将原式改写为差分形式：
$f(x) - f(x-1) = \sin\pi x$
分别将$x$替换为$x+1$、$x+2$，得到两个等式：
$f(x+1) - f(x) = \sin\left[\pi(x+1)\right] = \sin(\pi x + \pi) = -\sin\pi x$
$f(x+2) - f(x+1) = \sin\left[\pi(x+2)\right] = \sin(\pi x + 2\pi) = \sin\pi x$
将以上两式左右两边分别相加，左边中间项抵消：
$f(x+2) - f(x) = -\sin\pi x + \sin\pi x = 0$
即$f(x+2)=f(x)$，故$f(x)$是以$2$为周期的周期函数。
【考点】差分累加法证明周期性，利用三角函数周期性抵消附加项
