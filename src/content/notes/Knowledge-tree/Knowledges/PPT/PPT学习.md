#  1.快捷键
## 2. shift键
### 1. shift + 鼠标左键 == ==水平/竖直 移动

### 2. shift + 鼠标拖拽==缩放

### 3. shift + 旋转  == 每隔15°一转

## 2. ctrl键
### 1. ctrl + 鼠标滚轮 ==缩放

### 2.ctrl + 鼠标拖拽   ==复制到任意位置

### 3. ctrl  + d  ==快速复制元素（包括距离和方向）

### 4. ctrl + m    ==新建页面（等同于在左边目录视图中点击enter）

### 5. ctrl + g ==组合（用于把不同的元素组合）

## 3.ctrl +shift
### 1. ctrl +shift + 鼠标移动 ==快速水平/竖直复制

### 2. ctrl + shift + alt +改变大小 ==按中心缩放

### 3. ctrl + shift+ c/v ==格式刷

### 4. ctrl+ shift + g ==解除组合（与ctrl + g  组合相反）

## 4. 其他组合键
### 1. alt ->N ->X -> H  ==快速插入文本框

### 2. alt ->N ->P -> D  ==快速在页面中插入图片
