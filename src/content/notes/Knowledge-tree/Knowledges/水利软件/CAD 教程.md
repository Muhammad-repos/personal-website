# CAD 教程

## 1. 基础设置

### 1.1 快捷键草图设置

按  `F12` 来控制动态输入框的显示与否，关闭 动态输入框，可以 减少卡顿，流畅一些。建议关闭

按 `F3` 可以打开/关闭 对象捕捉，建议打开。

`F10` 极轴 要打开

> [!note]
>
> 开启后光标不会限制方向，而是从上一个点延伸出虚线，与光标进行对齐和追踪，虚线还可以帮助光标找到与其他线条 / 虚线的交点，起到定点的作用。
>
> 虚线方向可进行自定义（`SE` 草图设置 - >极轴追踪）
>
> - 增量角：为每两根虚线之间的夹角；
> - 附加角：一个角度只对应一根虚线，可以设置多个角度值，适应特定制图需求。



`F11` 对象捕捉追踪 要打开

> [!note]
>
> 开启后，光标可以在捕捉点上停留，即可从该点延伸出虚线，与光标进行对齐和追踪。
>
>  = 需要关闭`F12`  = 才能做出使用F 3、10、11的功能

`F7` 栅格，建议关闭

`F9` 捕捉，要关闭

`F8` 正交模式开启/关闭，建议关闭。

> [!note]
>
> 开启后，光标的编辑方向会被限制在水平/垂直的轴向上，还可以利用`UCS`用户坐标，控制正交的方向，`UCS`空格两下可以恢复
> （`PLAN`空格两下，正视于当前坐标）

### 1.3 基础操作笔记

####  一、视图与选择基础操作

1. **平移视图**：按住鼠标滚轮，拖动视图

2. **缩放视图**：滚动鼠标滚轮，实现位置缩放 / 全部显示

3. 双击鼠标滚轮，可以把它显示到刚好满屏。（即太大了会变小，太小了会变大。反正刚好满屏）

4. **三维围绕**：按住键盘`Shift`键，再按住鼠标滚轮拖动

5. **恢复平面**：左上角选择俯视，或者输入`PLAN`空格两下

6. **选择图形**：

   - 点选、框选（实线）、栏选（虚线）
   - 框选：从左往右 → 只选全在框内对象
   - 栏选：从右往左 → 碰到框就选择上
   - 按住拖动：套索选择工具
   - 全选：`Ctrl+A`

   

7. **取消选择**：减选（按住`Shift`键）、全部取消（`Esc`键）

8. **回车 / 空格**：执行指令、确定数值、重复上一个操作

9. **右键菜单**：常用功能菜单

####  二、通用系统快捷键

- `Ctrl+Z` 撤销 / `Ctrl+Y` 重做
- `Ctrl+S` 保存（良好习惯）
- `Ctrl+C/V` 复制 / 粘贴
- `Ctrl+Tab` 快速切换已打开图纸
- `Ctrl+1/2/3` 分别为特性栏、设计中心、工具选项板
- `Ctrl+8` 快速计算器
- `Ctrl+9` 隐藏 / 显示命令行

####  三、CAD 快捷指令核心特点

- 属于 Auto CAD 软件特色，可以做到左手键盘右手鼠标的盲打快捷画图操作，几乎所有功能都能以输入命令指令的方式执行。
- 输入指令后，按空格键，即可开始执行指令操作；中途输入数据和切换功能也是用空格键 / 回车键确认。
- 输入内容会显示在下方命令行中，可以鼠标点击，也可以键盘输入。
- 括号中的字母实现指令功能的变换。
- 有些指令不会自行中断，可以主动按空格键或者`Esc`键，中断指令。

##### 快捷键解释

1.  `L` 直线指令：

   - 鼠标点击即可产生线条，两个点以上时可以C空格直接闭合。

   - 执指令后，不点击直接空格会衔接到上个线条端点 。因此并且可以和圆弧A指令快速衔接（相切）。

   - C闭合功能：
     - 属于二级指令，在线条类指令中，符合条件时，
       才会出现（如 `ML`多线、`PL`多段线、`SPL`样条曲线、L直线）

2. `PL` 多段线指令：

   - 基本用法与L直线一致，只不过最终得到的线条为粘连在一起的**整体线条**，不会分段；可以使用X分解成直线

   - 二级命令：
     - W宽度：可以指定当前线段的头+尾线宽；
     - H半宽：与W宽度用法一致，只输入一半数值；
     - A圆弧/直线：同一段多段线，可以切换圆弧/直线模式绘制；
     - L长度：线性增加长度，不会转向；

3. `REC`矩形指令：
   - 绘制：先指定第一个点，再输入D空格（尺寸），先输入长度，再输入宽度，最后光标控制矩形的朝向，点击即可绘制出矩形；
   -  形状修改：在定点之前，有C倒角、F圆角等功能使用，其中标高、宽度、厚度是不常用的，不用学；
   - R角度：可以赋予矩形绘制时的角度值；
   - A面积：先输入面积再输入其中一条边长；
4. `M` 移动指令：
   - 输入`M` ，空格，选择对象，再空格确定，指定基点（移动点）放置到目标位置即可。
5. `TK`连续追踪点 ：*还没掌握*
   - 属于辅助指令，不能单独使用，要在其他指令执行中，需要定点定位的时候，输入`TK`空格，然后点击出发点，光标控制方向，输入追踪距离到达目标位置后，空格或者回车结束追踪；
   - 追踪效率提升：`TK`追踪方向会自动转弯，朝向鼠标位置，所以可以点击出发点后，光标直接扔到目标位置附近，然后连续输入距离，即可快速定位
6. `O`偏移指令：
   - 基础用法：先指定偏移距离，再选择对象，光标控制方向，点击即可产生线条；
   - 两种指定偏移距离的方法：
     - 1.直接输入偏移数值；
     - 2.用鼠标拾取图纸中两点之间的距离，可以连续多次偏移，也可以M多个快速偏移等距；
   - T通过：若不知道偏移距离，可以使用该功能，由光标确定偏移到的位置点；
   - E删除：偏移后是否删除原位置对象，默认不删；
   - L图层：决定偏移出的新线条归属图层（不常用）；
7. `C`圆形指令：
   - 基本画法：指定圆心，输入半径（D直径）；
   - `3P`三点画圆、`2P`两点画圆、T切点切点半径画圆；
     以及图标下拉条的三相切画圆（tanx3）
     第二个圆@空格可以快速绘制同心圆
   - 角度替代功能：
     在需要定点定方向的指令中，输入小于号和
     角度值，例如<60、<30，即可将光标的编辑
     方向限制在改角度方向上；
     - 角度值的输入，遵循“逆正顺负”规律
8. `TR`修剪指令：
   - 新版CAD ：TR空格一下，即可开始修剪，如需改回标准模式，可以TR空格O空格，输入S空格即可改回
     标准模式；
   - 简易修剪：适用于图形线条较少的情况输入TR空格下，哪里不要点哪里；
   - 常规修剪：先选择剪切边（起点终点的线条）输入TR空格一下，开始修剪图形；
9. EX延伸指令
   - 操作和原理，均与TR修剪指令一致，并且和TR修剪指令具有互通性；
   - 可以在修改图形时，按住shift键，实现切换，无需重新输入指令；
